# SoftBox
SoftBox — это веб-приложение, созданное для удобного хранения и распространения бесплатных утилит для операционной системы Windows. Платформа позволяет пользователям находить полезные программы для оптимизации, очистки и ускорения работы компьютера, а администраторам — эффективно управлять каталогом.
